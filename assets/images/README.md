# images/

Place your image assets here. Supported formats: `.svg`, `.png`, `.jpg`, `.webp`

## Suggested files

| Filename         | Description                              |
|------------------|------------------------------------------|
| `favicon.svg`    | Site favicon (SVG recommended)           |
| `og-image.png`   | Open Graph / social share thumbnail      |
| `logo.svg`       | App logo (optional)                      |
| `hero-bg.webp`   | Hero section background (optional)       |

## Notes

- Favour **SVG** for icons and logos (crisp at any size, small file).
- Use **WebP** for photos — great quality at smaller sizes.
- Optimise images before committing (tools: [Squoosh](https://squoosh.app), [SVGO](https://jakearchibald.github.io/svgomg/)).
